QUIQQER Währungen
========

Integriert verschiedenen Währung in QUIQQER


Paketname:

    quiqqer/currency


Features (Funktionen)
--------

- Automatische Aktualisierung der Wechselkurse
- Währungsumrechnung zwischen verschiedenen Währungen
